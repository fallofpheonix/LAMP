# README for El-Bagawat Brief 1 -- Pathway Tracing

## File Manifest

### Building Footprints
*ESRI Shapefile. Elevation information included. Each building is numbered.*
./BuildingFootprints.cpg
./BuildingFootprints.dbf
./BuildingFootprints.prj
./BuildingFootprints.shp (ESRI compliant shapefile)

### Point Marks
*ESRI Shapefile. Each point marks a terminus for path tracing.* 
*The points are located near the entrance of the marked building. For more*
*information on the locations of building entrances see Buildings.dwg and*
*Site_Map_With_ROI.png.*
./Marks_Brief1.cpg
./Marks_Brief1.dbf
.Marks_Brief1.prj
./Marks_Brief1.shp (ESRI compliant shapefile)
./Marks_Brief1.shx (ESRI compliant shapefile exchange format)

./DEM_Subset-Original.tif.aux.xml
./DEM_Subset-Original.tif
./DEM_Subset-WithBuildings.tif.aux.xml
./DEM_Subset-WithBuildings.tif
